Pipeline Description

The GenomeBoard Consilium pipeline performs multi-agent variant interpretation integrating ACMG/AMP 2015 classification criteria with Korean population-specific frequency data from the Korea Reference Genome Database (KRGDB v3.0). Each variant undergoes parallel evaluation by three AI-assisted consilium agents:

  • Clinical Agent — queries ClinVar for significance calls, associated conditions, and review star ratings
  • Korean Population Agent — compares allele frequencies between KRGDB and gnomAD (EAS/global), flags variants with ≥2x enrichment in Korean cohorts
  • Pharmacogenomics Agent — cross-references CPIC and PharmGKB guidelines with Korean-specific prevalence adjustments

Final classification follows ACMG/AMP 2015 criteria with ClinGen SVI updates. Korean enrichment ratio is calculated as KRGDB frequency / gnomAD global frequency.
